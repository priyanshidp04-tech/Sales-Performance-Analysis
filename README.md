📊 Sales Performance Analysis – Alfido Tech

🚀 Project Overview

This project presents an end-to-end Sales Performance Analysis using the Superstore dataset. The goal is to uncover valuable insights into sales trends, regional performance, and product efficiency to support better business decision-making.

🎯 Objectives

✔ Analyze sales across regions
✔ Identify top & low-performing products
✔ Understand category contribution
✔ Detect seasonal trends
✔ Provide actionable business recommendations

📂 Dataset Information

The dataset includes:

🧾 Order Details (Order ID, Order Date)
👥 Customer Information
🌍 Location (Region, State, City)
🛍 Product Details (Category, Sub-category, Product Name)
💰 Sales Data

⚠️ Note: Profit data is not available, so analysis focuses on revenue-based metrics.

🧹 Data Preparation
Removed extra spaces from columns
Converted date fields into datetime format
Handled missing values
Extracted time-based features (Year, Month)
📊 Key Metrics (KPIs)
Metric	Description
💰 Total Revenue	Total sales generated
📦 AOV	Average Order Value
📈 Key Insights
📅 Sales Trends
Peak: November – December
Low: January – February

👉 Strong seasonal demand pattern

🌍 Regional Performance
Uneven sales distribution across regions
High-performing regions dominate revenue

👉 Need for region-focused strategies

🛍 Category Performance
Technology is the top-performing category
Other categories lag behind

👉 Focus on high-performing segments

🏆 Product Insights
Few products drive majority of revenue
Many products underperform

👉 Follows 80/20 rule (Pareto Principle)

⚠️ Challenges
No profit data available
Uneven regional performance
Dependency on few products
Seasonal fluctuations
💡 Business Recommendations
🔥 1. Boost Top Products

Increase marketing & visibility

❌ 2. Optimize Weak Products

Discount / bundle / remove

🌍 3. Regional Strategy

Target low-performing regions

📅 4. Seasonal Planning

Stock & campaigns before peak season

💰 5. Increase AOV

Bundling & cross-selling strategies
